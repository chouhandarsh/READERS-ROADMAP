# READERS-ROADMAP
Build an Book recommedation webiste with search by mood  and a chat-bot for book recommendation in HTML and CSS using LLM
Readers Roadmap is a modern, interactive web application that helps users discover books based on genres, moods, ratings, trends, and conversational AI assistance. It provides a personalized and engaging reading discovery experience through an intuitive UI and smart filtering.
